# cc无限画布 v0.1.13

## 更新内容

- 优化文字标注宽度限制：文字框最大宽度从 1600px 放宽到 4096px，更适合长段剧本、提示词和宽版排版。
- 升级画布助手对“生图节点 / 图生图节点 / 视频节点 / 文字标注”等画布术语的理解，并增加创建生图节点的兜底计划。
- 助手聊天记录按画布自动备份到本地，重新打开画布后可继续之前的对话。
- 助手消息支持复制按钮、右键复制，以及选中文本后的 Ctrl+C 复制。
- 助手新增本地文件导入能力，可配合文字要求一起发送，用于分析剧本、文档和素材说明。
