# cc无限画布 v0.1.16

## 更新内容

- 优化画布助手计划识别：支持多段 JSON 计划合并，支持 `type/function/name/parameters/tool_calls` 等多种模型返回格式。
- 修复助手只返回“已识别到 N 个画布操作”时无法应用的问题；会根据上一条用户输入补全本地可执行计划。
- 新增计划“不使用”状态，避免误应用不想执行的助手计划。
- 助手聊天保留普通说明文字，同时隐藏可执行 JSON 代码块，历史代码块也会在加载时迁移为计划卡片。
- 设置页新增助手模型专用 Key 入口，可按当前助手模型单独配置 Key，未填写时回退平台总 Key。
- 修复图片/视频缩放、选区缩放、文字字号输入时输入一半就被强制应用的问题。
- 修复文字标注颜色编辑：保留原生取色器，新增常用色块调色板，支持手动输入 `#RRGGBB`、`RRGGBB`、`#RGB` 颜色代码。
- 本地静态文件改为不缓存，减少开发版或更新后仍加载旧前端脚本的问题。
