# cc infinite canvas v0.1.2

本次版本重点补齐桌面版 ChatGPT 联动、节点复制粘贴和安装器升级体验。

## 新增

- 新增桌面版启动器，安装后默认通过桌面版打开，保留浏览器版备用入口。
- 新增 ChatGPT 官网节点，可在无限画布中嵌入 ChatGPT 官方网页使用。
- ChatGPT 节点支持单独配置代理，仅作用于该节点的登录和使用。
- ChatGPT 节点支持第三方登录弹窗，Google、Microsoft、Apple 等登录方式会继承同一代理和登录会话。
- 全部节点支持 `Ctrl+C` 复制、`Ctrl+V` 粘贴，支持多选节点一起复制粘贴。
- 安装器会读取旧版本安装路径和缓存路径，默认覆盖安装到原位置。

## 优化

- ChatGPT 节点窗口大小不再受画布缩放影响，放大或缩小画布时窗口保持屏幕尺寸稳定。
- ChatGPT 节点增加“拖动”按钮，交互网页和移动节点的操作更清晰。
- 安装包内置 Electron 桌面运行时，安装后不需要再单独执行 `npm install`。
- 图形安装器优先使用带窗口的安装界面，减少只看到后台 PowerShell 的情况。

## 修复

- 修复 ChatGPT 节点选中时容易跟随鼠标误移动的问题。
- 修复安装器无法自动沿用旧版本安装位置的问题。

## 注意

- 因为安装包现在内置桌面运行时，安装包体积会明显变大。
- 如果旧版本正在运行，覆盖安装前建议先关闭旧窗口。
