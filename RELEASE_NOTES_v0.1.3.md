# cc infinite canvas v0.1.3

本次版本修复版本更新弹窗的异常显示问题。

## 修复

- 修复点击“检查更新”时，接口返回纯文本错误会显示 `Unexpected token` 的问题。
- 修复配置尚未读取完成时，版本更新弹窗里“当前版本”容易显示为 `-` 的问题。
- 后端更新检查增加 GitHub 请求超时和网络错误兜底，错误会统一返回 JSON，前端可以正常展示。

## 验证

- 已验证 `/api/config` 可以返回当前版本号。
- 已验证 `/api/update/check` 可以返回 GitHub Release、最新版本和安装包信息。
