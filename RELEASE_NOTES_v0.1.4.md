# cc infinite canvas v0.1.4

## 新增

- 启动应用后自动检查 GitHub Releases；仅在发现新版本时弹出更新提示，不强制更新，可直接关闭。
- 新增 Grsai 平台 API Key 配置入口，并支持 `nano-banana-2` 生图模型。

## 改进

- 打开设置或版本更新弹窗时，临时隐藏桌面端 ChatGPT 视图，避免遮挡弹窗内容。
- 生图成功后自动清理生图任务节点；生图失败时保留节点，方便继续修改重试。
- 点击提示词复用时，会同时恢复生成图片使用的提示词、模型参数和参考图信息。
- 文字节点字体大小上限提高到 240。
- 安装包继续同时包含网页版启动器和桌面版启动器，桌面与主开始菜单快捷方式优先指向桌面版。

## 打包

- 本地数据目录不会打进安装包，包括 `cache/`、`outputs/`、`data/` 和本机 `.env.local`。
