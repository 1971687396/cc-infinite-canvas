# cc infinite canvas v0.1.5

## 改进

- 桌面版不再依赖固定的本地 `3000` 端口。Electron 会在应用进程内启动服务，并由系统自动分配空闲端口。
- 桌面版不需要单独启动或保持浏览器页面，关闭桌面应用时内置服务会同步退出。
- 本地服务仅监听 `127.0.0.1`，避免暴露到局域网。
- 直接运行 `npm start` 时，如果配置端口已被占用，服务会自动选择其他空闲端口并输出实际地址。
- 网页版启动器会提前检查配置端口；端口冲突时自动切换，并在服务就绪后再打开浏览器。

## 打包

- 安装包继续同时包含桌面版和网页版启动器，桌面快捷方式优先指向桌面版。
- 本地 `cache/`、`outputs/`、`data/` 和 `.env.local` 不会进入安装包。

## 验证

- 已在 `3000` 端口被占用的情况下验证自动回退，服务成功选择其他端口且 API 正常响应。
