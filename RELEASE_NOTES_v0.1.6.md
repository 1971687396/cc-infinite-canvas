# cc infinite canvas v0.1.6

## 新增

- 统一生图节点加入 Dreamina（即梦）CLI 模型，支持文生图、图生图、模型版本和比例/分辨率切换。
- 图生图模式支持最多 10 张参考图，结果会按当前画布缓存目录保存。
- 设置页新增即梦 CLI 安装、登录、测试连接按钮，用户可以直接从应用内启动官方流程。

## 改进

- 即梦登录改为官方 `dreamina login` 流程，避免授权链接打开成新画布页面。
- Dreamina 结果保留 CLI 返回的真实像素尺寸，方便在无限画布中排版。
- 设置页保留复制安装/登录命令作为兜底，并显示 CLI 版本、登录状态、余额和会员等级。

## 验证

- 已通过 Node 语法检查、依赖检查和 Git diff 空白检查。
- 已验证本机 Dreamina CLI 状态读取和余额读取；未额外提交新的付费生图任务。
