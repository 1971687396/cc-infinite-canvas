# cc无限画布 v1.0.0

`1.0.0` 是 cc无限画布的首个正式版本。本次更新集中完善模型平台兼容、工程可靠性、Photoshop 工作流和桌面端创作体验。

## 多平台模型连接

- 每个生图和对话模型都可以独立选择平台、API Key、API 模型 ID、Base URL、接口路径、协议与鉴权方式。
- 内置云雾、OpenAI、Google Gemini、Anthropic、xAI、DeepSeek、火山方舟和 Grsai 预设，并支持自定义中转站。
- 支持 OpenAI Images、OpenAI Chat Completions、Gemini `generateContent`、Anthropic Messages 与 Grsai 异步生图协议。
- 模型专用 Key 优先于全局备用 Key，旧版云雾配置会自动迁移。
- 生图节点默认跟随模型连接，也可以在高级设置中为单个节点覆盖 API 地址。

## Photoshop 联动

- PS Image AI 新增独立 Agent 工作流，可复用画布的对话模型配置，不会把明文 Key 下发给插件。
- Agent 可以读取 Photoshop 文档与图层上下文，生成白名单操作计划，确认后再执行并支持一次撤销。
- 修复 Agent 模型列表重复、模型设置入口缺失与计划文字重叠问题。
- 设置页新增 PS 插件下载入口，可直接前往最新 GitHub Release。
- Photoshop 桥接继续使用固定本地端口，不受画布界面随机端口影响。

## 工程与数据可靠性

- 工程保存改为原子写入，并保留待提交文件和上一份有效备份。
- 启动时会比较磁盘工程、浏览器快照与备份，自动恢复停电或异常退出前的最新完整数据。
- 浏览器快照缺少磁盘索引时会重新写回画布列表，降低工程“缓存还在但入口消失”的风险。

## 创作体验

- 优化浅色和深色模式、悬浮工具栏、设置页、节点卡片与助手面板的布局和可读性。
- 补齐本地随机肖像词库，并保留随机种子、逐项锁定、中英文提示词和节点连线能力。
- 即梦 CLI 支持应用内安装/更新、登录、切换账号、动态读取生图模型，并增强混杂日志响应的解析兼容。
- 助手与模型连接支持 `gpt-5.6-sol` 等新增文本模型。

## 安装说明

- Windows 安装包：`YunwuImageCanvasSetup.exe`
- 支持覆盖安装并沿用旧版安装位置和缓存位置。
- 安装包不包含本地 API Key、画布缓存、生成结果或开发依赖。
