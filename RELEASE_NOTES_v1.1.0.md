# cc无限画布 v1.1.0

`1.1.0` 的核心更新是正式接入火山方舟图片与视频生成能力，并完善方舟模型的独立配置、尺寸校验和画布工作流。

## 火山方舟

- 设置中新增独立的“火山方舟”页面，可配置方舟 API Key、Base URL，以及每个模型实际使用的 Model ID。
- 新增 Seedream 5.0 Pro、Seedream 5.0 Lite、Seedream 4.5 和 Seedream 4.0 生图模型。
- 支持方舟文生图、单图及多图参考、提示词优化、输出格式和连续组图参数。
- 新增 Seedance 2.0、Seedance 2.0 Fast 和 Seedance 2.0 Mini 视频模型。
- 支持视频参考图、时长、比例、分辨率、音频及模型高级参数，并自动轮询任务和缓存生成结果。
- 方舟模型也可使用模型专用 Key；未填写时回退到方舟平台 Key。

## 尺寸兼容

- 按不同 Seedream 型号分别提供官方支持的 1K、2K、3K、4K 档位和常用比例尺寸。
- 增加服务端总像素、宽高比和型号范围校验，避免把无效尺寸提交到方舟接口。
- 旧版 Seedream 5.0 Pro 的 `1424x800` 等不兼容尺寸会按原比例迁移到合法尺寸，例如 `2816x1584`。
- 修正部分 3K、4K 和 21:9 尺寸与方舟限制不一致的问题。

## 视频工作流

- 生成完成的视频新增“复用提示词”，可完整复用模型、比例、分辨率、时长、参考图和高级参数。
- 多选视频结果与视频任务节点时，可直接将视频生成配置应用到所选任务。
- 画布选区工具栏现在同时支持复用图片和视频提示词。

## 安装说明

- Windows 安装包：`YunwuImageCanvasSetup.exe`
- 支持覆盖安装并沿用旧版安装位置和缓存位置。
- 安装包不会包含本地 API Key、画布缓存、生成结果或开发依赖。
