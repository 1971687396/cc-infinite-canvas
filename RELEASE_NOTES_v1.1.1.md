# cc无限画布 v1.1.1

`1.1.1` 增加火山方舟 AIGC 虚拟人像素材入库工作流，并优化画布选图、素材连线和视频提示词的精确引用体验。

## 方舟素材入库

- 新增“虚拟人像入库”节点，可连续点击画布图片，也可通过可视连线添加图片。
- 支持使用 Access Key ID 与 Secret Access Key 调用方舟素材接口。
- 仍保留公网地址的图片可直接登记；只有本地文件的图片会先上传到配置的 TOS 存储桶。
- 自动创建或复用 AIGC 素材组，并轮询素材状态直到可用或失败。
- 单张图片失败不会中断同一节点里的其他图片。

## 可视素材工作流

- 入库结果会直接回写到原图片，显示“方舟处理中”“方舟素材”或“入库失败”标记。
- 带“方舟素材”标记的图片连接到方舟视频节点时，会自动使用认证素材，不需要手动复制 `asset://` 标识。
- 原图片、入库节点与视频节点之间保留可视连线，方便确认素材来源和使用位置。
- 普通参考图片与方舟认证素材合计最多 9 个，并在添加时自动校验数量。

## 视频素材引用

- 视频提示词输入 `@` 会弹出当前节点已添加的参考素材，显示缩略图、名称与输入顺序。
- 支持鼠标选择，以及方向键、Enter 和 Tab 快速插入。
- 已引用素材会在输入框下方显示独立标签，避免和普通提示词粘连。
- 提交任务时会把 `@素材` 转换成带素材序号和名称的明确边界描述，帮助模型精确定位参考素材。

## 安装说明

- Windows 安装包：`YunwuImageCanvasSetup.exe`
- 支持覆盖安装并沿用旧版安装位置和缓存位置。
- 安装包不会包含本地 API Key、Access Key、Secret Key、画布缓存或生成结果。
