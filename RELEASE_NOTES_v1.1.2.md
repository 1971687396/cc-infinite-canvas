# cc无限画布 v1.1.2

`1.1.2` 完善火山方舟角色素材的跨设备工作流，并修复云雾平台 `gpt-image-2` 纯文生图出现 Unknown model 的兼容问题。

## 角色素材组

- “虚拟人像入库”节点现在以角色为单位选择或新建素材组，不同角色可分别管理。
- 支持读取方舟素材组列表，并在节点内按名称筛选和复用已有角色组。
- 同一节点可集中保存一个角色的不同角度、表情和妆造，减少素材混用。

## 跨设备素材库

- 支持从方舟素材库同步指定角色组的已有素材，并以缩略图形式浏览。
- 云端素材可下载到当前画布缓存，自动恢复方舟素材标记，无需在新设备上重复入库。
- 支持将云端素材身份绑定到当前唯一选中的同一张本地图片，方便迁移旧工程。

## 生图兼容修复

- 修复云雾平台 `gpt-image-2` 在纯文生图接口返回 `Unknown model` 时无法生成的问题。
- 仅在确认是云雾预设、纯文生图且错误模型完全匹配时，自动改用兼容路由 `gpt-image-2-c` 重试。
- 图生图、官方 OpenAI 和用户自定义接口保持原有调用方式，不受该兼容逻辑影响。

## 安装说明

- Windows 安装包：`YunwuImageCanvasSetup.exe`
- 支持覆盖安装并沿用旧版安装位置和缓存位置。
- 安装包不会包含本地 API Key、Access Key、Secret Key、画布缓存或生成结果。
