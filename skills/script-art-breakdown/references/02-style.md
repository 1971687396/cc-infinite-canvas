风格模块｜真人3D2D漫画游戏CG
用于判断与统一真人、3D、2D、漫画、游戏CG、绘本、水墨等视觉风格。

# 一、使用场景

当用户指定视觉风格，或需要根据剧本推荐视觉风格时，调用本模块。不要默认真人剧；所有人物、场景、道具提示词都必须统一到目标视觉风格。

# 二、风格判断规则

用户指定真人剧：使用真实摄影、真实材质、真实布光、真实皮肤纹理。
用户指定3D动漫：使用二次元3D模型、卡通渲染、清晰轮廓、发丝块面、材质分区。
用户指定3D国漫：使用高精度国漫角色、东方幻想美术、修长比例、华丽服装层次。
用户指定3D写实CG：使用PBR材质、电影级灯光、真实反射、3D资产结构。
用户指定2D日漫：使用清晰线稿、赛璐璐上色、干净色块、动漫眼睛。
用户指定2D国漫厚涂：使用半写实动漫脸、细腻光影、东方纹样、厚涂材质。
用户指定漫画风：使用线稿、黑白灰、网点、排线、分镜感。
用户指定游戏CG：强调装备结构、材质分层、强角色剪影、游戏资产感。
用户指定儿童绘本：强调温暖色彩、纸张纹理、可爱亲和、低攻击性。
用户指定低多边形：强调low poly、几何块面、简洁色块。
用户指定水墨国风：强调宣纸纹理、水墨晕染、淡彩点染、东方意境。

真人剧 / 写实摄影：真人实拍影视风格，电影级真实摄影质感，自然光影，真实材质，人物和场景均符合现实拍摄逻辑，适合真人剧、短剧、电影美术资产制作。

# 五、风格一致性检查

用户指定2D时，不要写PBR材质和真实毛孔。
用户指定3D时，不要只写平面插画和纸张纹理。
用户指定真人时，不要写动漫大眼、赛璐璐、Q版比例。
用户指定国漫时，不要写成日漫少女风。
用户指定日漫时，不要写成国漫厚涂。
用户指定低多边形时，不要堆真实复杂纹理。
用户指定儿童绘本时，不要成人化、性感化、暗黑化。
同一批人物、场景、道具必须统一风格，不要混用。
