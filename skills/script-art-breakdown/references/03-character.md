人物模块｜脸型五官服装身份年龄规则
用于角色、人设、脸型、五官、身材、服装、发型、定脸图、服装图、多视图提示词。更新版已加入分阶段工作流配合规则与母版人物资产规则。

# 一、使用场景

当任务需要分析人物、人设、脸型、五官、身材、服装、发型、配饰、定脸图、服装图、多视图、母版人物资产、换装防变脸时，调用本模块。

# 二、分阶段配合规则

第一阶段美术定调时，人物模块只输出“人物美术方向初判”，不输出完整提示词。需要判断主角、反派、配角、群演的视觉方向、颜值等级、身材比例、服装复杂度、妆发方向、是否需要母版人物资产、哪些人物需要多套造型。
第三阶段资产拆解时，人物模块输出人物资产总表和人物详细设定，可以先只输出设定，不必输出所有生图提示词。
第四阶段提示词生成时，才输出母版人物资产提示词、定脸图提示词、服装设定图提示词、多视图提示词、多套服装换装提示词。

# 三、人物设计优先级

第一优先级：用户明确指定。用户给参考图、服装、脸型、五官、风格、禁止元素时，严格执行。
第二优先级：剧本明确描写。剧本写身份、职业、年龄、场合、状态时，必须遵守。
第三优先级：根据人物身份、年龄、职业、阶层、性格、剧情功能推导。
第四优先级：根据题材和视觉风格补足。

# 四、每个人物必须分析

姓名、性别、年龄段、身份职业、阶层地位、与其他人物关系、剧情功能、首次出场场景、是否需要多套造型、是否需要状态版本。
脸型、五官、眉眼、鼻型、嘴唇、皮肤或上色质感、骨相、神态、妆容、记忆点。
身高感、头身比例、肩颈、腰线、体态、年龄感、力量感或脆弱感。
服装版本、主色调、辅助色、点缀色、材质、新旧程度、内层、中层、外层、领口、袖型、腰部结构、下装或衣摆、鞋履、发型、发饰、配饰。
目标风格下的表现方式：真人、3D、2D、漫画、游戏CG等。

# 五、人物状态版本

日常、正式、受伤、落魄、黑化、战斗、婚服、官服、便服、夜行服、囚服、病号服、舞台服、校服、职场服、礼服、实验服、机能服、战甲、战损、老年、少年、伪装、复仇后、身份揭露后、魔化、神化、觉醒、变身、怪物化、Q版、幼年回忆版。

# 六、脸型五官设计规则

主角：不能普通路人化。男主需要骨相清晰、眉眼有记忆点、鼻梁挺直、下颌线清楚、身材比例修长、气质符合身份；女主需要脸型柔和但有记忆点，眼神能承载情绪，五官清晰，妆容贴合人物阶段。
反派：不要脸谱化，可根据类型设计为窄长脸、锋利眉眼、薄唇、冷白或暗沉肤色、眼神压迫、骨相硬、表情克制、贵气反派、阴鸷反派、油腻反派、斯文败类、武力型压迫反派。
权贵人物：脸型端正，骨相优越，神态稳定，眼神不急不躁，妆发规整，皮肤状态更精致，服装材质更高级，配饰克制但贵重。
底层人物：皮肤粗糙，晒痕，皱纹，疲惫感，手部粗糙，服装旧但合理，身材自然，不要精致模特感。
少年 / 少女：面部轮廓更柔和，骨相未完全成熟，眼神干净，妆容更淡，服装不能成人化过度，身材比例自然修长。

# 七、目标风格下的脸型五官处理

真人剧：强调脸型、眉眼、鼻型、嘴唇、皮肤质感、骨相、真实年龄感，皮肤有真实纹理，毛孔自然，五官清晰，不过度磨皮。
3D动漫：二次元脸型、眼睛比例、鼻口简化程度、发丝块面、干净皮肤、清晰轮廓线，避免恐怖谷和真人毛孔过多。
2D动漫：线稿粗细、眼睛形状、表情符号化程度、上色方式、发丝线条、脸部阴影，五官更概括，线条更清晰。
2D国漫厚涂：半写实五官、光影结构、皮肤色块、五官精致度、材质笔触，适合立绘和高完成度概念图。
Q版：头身比、脸型圆度、眼睛大小、表情夸张度、服装简化程度。
游戏CG：英雄化比例，肩腰腿线条更明确，装备承重合理。

# 八、年龄段服装规则

儿童，3到12岁：舒适、方便活动、颜色明亮或柔和、款式不成人化。适合T恤、卫衣、背带裤、连衣裙、校服、儿童睡衣、运动鞋。避免高跟鞋、成人礼服、浓妆、性感化。
少年 / 少女，13到18岁：校园、家庭、运动、日常场景为主。适合校服、白衬衫、针织开衫、运动外套、牛仔裤、百褶裙、连帽卫衣、书包、球鞋。避免成人化、高奢化、浓妆、过度性感。
青年，19到30岁：大学生可用卫衣、牛仔裤、衬衫、帆布鞋、双肩包；初入职场用基础衬衫、西裤、针织衫、简单外套、通勤包；白领用衬衫、西装裤、半裙、风衣、乐福鞋；总裁精英用高定西装、羊毛大衣、真丝衬衫、皮鞋、腕表；普通打工人使用基础款外套、T恤、牛仔裤、帆布鞋并保留通勤疲惫感。
古装青年：少年文臣可用月白、青灰、浅蓝、直裾或圆领袍；贵族公子可用锦缎、暗纹、玉佩、发冠；王爷 / 世子可用深色外袍、金线暗纹、宽肩结构、玉带、靴子；少女小姐可用襦裙、披帛、发簪、珠钗，色彩按性格区分。
成年人，31到45岁：强调职业、家庭责任、经济状态。职场中层用西装外套、衬衫、针织衫、风衣；公司高管用高级西装、羊绒大衣、真丝衬衫、腕表；普通父母用家居服、旧外套、围裙、舒适鞋。古装权臣可用深色官服、补子或暗纹、玉带、官帽；古装主母 / 贵妇用稳重色、绸缎、高发髻、克制贵重饰品。
中年，46到60岁：强调社会阅历、经济状态、生活痕迹。成功商人用深色夹克、西装裤、皮鞋、腕表；暴发户可用亮面材质、金色配饰、品牌感强、颜色张扬；普通母亲用针织衫、棉麻上衣、围裙、舒适裤装；古装家主用深色锦袍、厚重外袍、玉扳指、腰带；古装主母用深色绸缎、端正发髻、金簪玉钗。
老年，60岁以上：舒适、保暖、实用。普通老人用毛衣、棉服、布鞋、旧围巾、手杖、老花镜；富贵老人用羊绒衫、唐装、玉饰、手杖、精致眼镜；病中老人用病号服、开衫、毯子、拖鞋、虚弱感；古装老太君用深色华贵外袍、厚重披肩、金玉发饰。

# 九、身份职业服装规则

现代身份：总裁 / 企业继承人强调高级、利落、克制、权力感，使用高定西装、羊毛大衣、真丝衬衫、皮鞋、腕表。职场白领强调通勤、干净、现实，使用衬衫、西裤、半裙、针织衫、通勤包。律师 / 金融 / 咨询强调严谨、理性、精英，使用深色西装、白衬衫、公文包、皮鞋、金属眼镜。医生 / 护士强调干净专业，使用白大褂、洗手服、护士服、听诊器、工牌。警察 / 刑警 / 法医强调职业、克制、疲惫、行动方便，使用警服、夹克、工装裤、证件夹、手套、防护服。学生使用校服、运动外套、书包、球鞋，年龄准确。网红 / 明星需要舞台服、私服、红毯、直播间服装，镜头前后有反差。底层打工人使用工作服、反光马甲、旧T恤、工装裤、劳保鞋、汗渍、灰尘、磨损。
古装身份：皇帝 / 帝王使用冕服、龙纹、玉带、朝靴、冕冠、厚重外袍。王爷 / 世子 / 贵族公子使用锦缎外袍、暗纹、玉佩、宽腰封、发冠、靴子、金线或银线点缀。文臣 / 谋士 / 书生使用圆领袍、直裰、长衫、宽袖、腰封、玉佩、发冠或木簪。武将 / 护卫 / 士兵使用甲胄、护腕、护肩、靴子、腰带、披风、窄袖、皮革或金属件。贵族小姐 / 公主使用襦裙、披帛、宽袖外衫、绣纹、发簪、步摇、珠钗、玉佩。丫鬟 / 仆人 / 侍从使用棉麻衣裙、简单腰带、窄袖、布鞋、简单发髻，无贵重配饰。歌姬 / 舞姬使用轻纱、披帛、亮色裙装、金银饰、发簪、额饰、舞鞋、飘带。百姓 / 商贩 / 劳工使用粗布衣、短打、围裙、头巾、布鞋、补丁、汗渍、灰尘、磨损。

# 十、场合服装规则

日常居家：舒适、放松、生活感。工作职场：职业识别、可信、效率、身份等级。商务谈判：权力、克制、专业、压迫感。宴会酒会：正式、社交、身份展示、视觉记忆点。婚礼大婚：仪式感、身份、情绪、剧情转折。战斗逃亡：行动方便、防护、紧张、实用。病中受伤落魄：虚弱、破损、剧情状态、情绪。祭祀朝堂审判：仪式、权力、秩序、压迫。

# 十一、主要人物母版资产规则

当主要人物满足以下任一条件时，必须先输出母版人物资产：主要人物有多套服装；主要人物有多个剧情阶段；主要人物跨集反复出现；主要人物需要后续频繁换装；用户要求“换装不变脸”“保持同一个人”“别变脸”；人物需要定脸图、服装图、多视图、战损图、婚服图、朝服图等多种资产。
母版人物资产用于固定脸型、五官、发型基础、头身比例、肩腰比例、气质记忆点。后续所有服装版本都必须基于母版进行换装，只更换服装、发饰、配饰、妆容强弱和剧情状态，不改变人物身份识别。

## 母版人物资产必须固定

脸型、五官比例、眉眼形状、鼻型、嘴唇、面部骨相、皮肤或上色质感、基础发型轮廓、头身比例、肩颈比例、腰线位置、身材气质、人物气质记忆点、角色身份识别点。

## 母版人物资产不能固定死

后续换装时可以改变服装、发饰、妆容强弱、配饰、外袍、鞋履、剧情状态、战损程度、情绪表情、光影氛围。

负面提示词：服装结构混乱，身份不符，年龄不符，场合不符，风格不统一，过度华丽，过度廉价，材质错误，颜色脏乱，腰间多余武器，奇怪配饰，低清晰度，比例五短，小短腿，头大身小，衣服糊成一片。

## 3. 人物多视图提示词

【用户指定视觉风格】，21:9超宽横版，角色多视图设定板。画面左侧三分之一为人物超大面部特写，完整呈现从头顶到肩膀的区域，清晰展示五官、脸型、发型、发饰、妆容、皮肤或上色质感。右侧依次排列角色正视图、侧视图、后视图，三者均为全身完整呈现，从头到脚完整入镜，比例统一，服装、发型、发饰、腰封、鞋履、配饰全部清晰。人物为【填写人物身份】，气质为【填写人物气质】，服装为【填写服装版本】，年代为【填写年代】，色调为【填写色调】，适合该风格的角色资产图，边缘清晰，无多余元素。

## 4. 多套服装换装提示词

基于已确认的【人物母版资产】，保持同一张脸、同一脸型、同一五官比例、同一基础发型轮廓、同一头身比例、同一肩腰比例和同一气质记忆点，仅更换为【服装版本名称】。服装根据【剧情场景】【人物状态】【身份阶层】设计，主色为【主色】，辅助色为【辅助色】，材质为【材质】，结构包括【内层】【外层】【腰封/腰带】【袖型】【鞋履】【配饰】。禁止改变人物年龄感、脸型、眉眼、鼻型、嘴唇、身材比例和身份识别。
负面提示词：变脸，五官漂移，脸型变化，年龄变化，身材比例变化，发型基础变化过大，气质变化，人物不像同一个人，服装与身份不符，服装结构混乱。

是否遵循用户指定。
