用于剧情关键道具、交互物件、白底资产图、道具特写、场景使用图。

# 一、使用场景

当任务需要分析道具、剧情关键物、人物手持物、场景陈设、交互元素、白底资产图、场景使用图时，调用本模块。

# 二、每个道具必须分析

道具名称、出现场景、使用人物、剧情作用。
年代背景、材质、颜色、新旧程度、尺寸比例。
是否需要特写，是否需要角色手持版本。
是否需要白底资产图，是否需要与场景风格统一。
是否是关键道具，是否有剧情信息。
在目标视觉风格下的造型复杂度。

# 三、道具类型库

古装道具：折扇、玉佩、香囊、发簪、令牌、卷轴、奏折、官印、茶具、药瓶、账本、马车、灯笼、拂尘、刑具、佩剑、匕首、酒壶、食盒。
现代道具：手机、电脑、文件袋、合同、病历、钥匙、银行卡、药瓶、照片、监控录像、U盘、行李箱、证件、录音笔、车钥匙、快递盒。
悬疑道具：旧照片、录音带、信件、钥匙、锁、案卷、证物袋、血迹布料、报纸剪报、监控硬盘、手套、遗物、指纹杯、破损手机。
科幻道具：控制台、通讯器、身份芯片、AI核心、数据终端、能量电池、武器模块、防护头盔、实验样本、机甲零件、星图、战舰徽章。
仙侠神话道具：法剑、符箓、玉牌、丹药、灵石、法器、神印、祭器、天书、神弓、仙铃、阵盘、灵兽项圈。
现实生活道具：饭碗、药盒、旧照片、存折、钥匙串、菜篮子、病历本、课本、书包、老家具、收音机、搪瓷杯、工牌。

# 四、道具设计原则

道具必须服务剧情，不要只做装饰。
道具必须贴合人物身份、时代背景、场景需要和视觉风格。
关键道具要有明确记忆点，例如材质、纹样、磨损、编号、印章、符文、血迹、封条、芯片、徽章等。
普通道具要真实自然，不抢戏，但要清晰可见。
道具新旧程度要符合剧情状态：新、旧、破损、血污、烧灼、潮湿、磨损、锈蚀、洁净、被封存等。
同一剧本中重复出现的道具必须保持造型统一。

# 五、目标视觉风格下的表现方式

真人剧：道具需要真实可制作，材质、磨损、比例合理，适合真人演员手持或场景交互。
3D动漫：道具需要适合建模，轮廓清楚，材质分区明确，不能过碎。
2D动漫：道具需要线稿清晰，色块简洁，便于动画复用。
游戏CG：道具需要有装备结构、功能分区和细节等级，适合做资产或交互物。
漫画风：道具需要黑白轮廓清楚，特征强，便于分镜识别。
儿童绘本：道具需圆润、友好、色彩温暖，避免危险感过强。
水墨国风：道具需有东方意境，轮廓简洁，材质可通过墨色和淡彩表现。

# 六、道具输出模板

### 道具编号：道具名称
#### 1. 剧情作用
说明它在剧情中起什么作用，是普通陈设还是关键道具。
#### 2. 使用人物与场景
说明由谁使用，出现在哪些场景。
#### 3. 造型设定
说明材质、颜色、纹理、新旧程度、尺寸、年代感、是否有文字或符号。
#### 4. 目标视觉风格下的表现方式
说明该道具在真人剧、3D、2D、漫画、游戏CG等目标风格下如何表现。
#### 5. 白底资产图提示词
输出单独道具资产图提示词。
#### 6. 场景使用提示词
输出道具在具体场景中出现和交互的提示词。

# 七、道具白底资产图提示词模板

【用户指定视觉风格】，16:9横版或1:1方图，纯白背景，影视 / 动画 / 游戏道具资产设定图，【道具名称】单独展示，无人物，无手持，正面角度和轻微三分之四角度结合，结构清晰，材质为【填写材质】，颜色为【填写颜色】，年代背景为【填写年代】，风格符合【填写人物/场景需求】，细节包括【填写关键细节】，边缘清晰，适合【真人道具制作 / 3D建模 / 2D动画资产 / 漫画设定 / 游戏资产】参考。
负面提示词：低清晰度，结构不清，材质错误，年代错误，比例怪异，多余文字，乱码，过度华丽，廉价塑料感，风格混乱，背景复杂，多余人物，多余手。

# 八、道具场景使用提示词模板

【用户指定视觉风格】，21:9横版，【场景名称】中出现【道具名称】，【道具】放置在【具体位置】，与【人物/场景】产生明确交互关系，材质、大小、颜色与白底资产图一致，光影符合场景环境，真实自然或风格统一，不抢戏但清晰可见，适合【特写 / 中景 / 动画镜头 / 漫画分镜 / 游戏交互镜头】。

# 九、关键道具检查清单

是否明确剧情作用。
是否明确使用人物与出现场景。
是否明确材质、颜色、新旧程度、尺寸比例。
是否符合时代背景和人物身份。
是否符合场景光影和目标视觉风格。
是否需要白底资产图。
是否需要场景使用图。
是否需要特写或手持版本。
是否避免多余文字、乱码、廉价塑料感。
